# 🛒 쇼핑 리스트 앱

HTML, CSS, JavaScript로 만든 심플한 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 완료 표시
- 개별 아이템 삭제
- 완료된 항목 일괄 삭제
- 총 아이템 수 / 완료 수 표시
- **localStorage**에 데이터 저장 (새로고침해도 유지)

## 사용 방법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다. 별도의 설치나 서버가 필요 없습니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage API
